Gebruik je een nginx/Caddy/NPM reverse proxy? Dan moet je CORS-headers in nginx instellen — de HA-instelling bereikt nginx niet.

  1. Zoek je nginx location / blok voor HA.
  2. Voeg deze regels toe:
add_header 'Access-Control-Allow-Origin' 'http://localhost:8080' always; add_header 'Access-Control-Allow-Headers' 'Authorization,Content-Type' always; add_header 'Access-Control-Allow-Methods' 'GET,OPTIONS' always; # Handle preflight if ($request_method = OPTIONS) { add_header 'Access-Control-Allow-Origin' 'http://localhost:8080'; add_header 'Access-Control-Allow-Headers' 'Authorization,Content-Type'; add_header 'Access-Control-Max-Age' 3600; return 204; }
  1. Reload nginx: nginx -s reload

💡 Hoe werkt dit?

Deze tool vergelijkt voor jouw eigen stroomverbruik twee soorten contract:

  • Vast contract — een vaste prijs per kWh (piek & dal), het hele jaar gelijk.
  • Dynamisch contract — de prijs volgt elk uur de stroombeurs (EPEX): 's nachts en bij veel zon/wind goedkoop, in de avondpiek duur.

⚠️ Dit is een 2027-model. Vanaf 1 januari 2027 vervalt de salderingsregeling — je mag teruggeleverde zonnestroom niet meer wegstrepen tegen je afname. Daarom rekenen we energiebelasting over je bruto afname en krijg je een aparte, lagere terugleververgoeding. Met de regels van vandaag zou de uitkomst er anders uitzien.

📊 Let vooral op het verschil tussen beide contracten — niet op de absolute bedragen (zie de toelichting bij de cijfers hieronder).

💶 Hoe wordt de dynamische prijs per uur berekend?

Bij een dynamisch contract verandert je kale stroomprijs elk uur. Wat je per kWh afneemt betaalt, is opgebouwd uit vier delen:

EPEX-beursprijs
dat uur + opslag
leverancier + 21% BTW
over die twee + energiebelasting
vast per kWh = all-in prijs

Rekenvoorbeeld (één uur, met de huidige instellingen):

Voorbeeld verschijnt zodra een simulatie is gedraaid…
  • Teruglevering wordt vergoed tegen de kale EPEX-prijs van dat uur (zonder opslag/EB). Bij een negatieve prijs betaal je juist bíj om terug te leveren.
  • Energiebelasting (2027): sinds het einde van de saldering betaal je deze over álle afgenomen kWh — je teruglevering wordt er niet meer van afgetrokken.

📋 De volledige uur-voor-uur-uitwerking staat in de tabel Dynamisch Contract Specificatie hieronder — klik daar op “Afname (EPEX + opslag + BTW)” voor de tabel per uur of per dag.
